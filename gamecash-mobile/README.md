# تطبيق الجوال الموحد لصالون ألعاب GameCash

مرحباً بك! هذا هو الكود المصدري الكامل لتطبيق الجوال الخاص بنظام **GameCash** لإدارة صالة ألعابك. تم بناء التطبيق بالكامل باستخدام إطار عمل **Flutter** ليعمل بسرعة فائقة، ومظهر جمالي فاخر يدعم الشاشات الداكنة والمؤثرات الزجاجية (Dark Glassmorphic UI).

## مميزات تطبيق الجوال

1. **شاشة البيع الموحدة والمباشرة (الواجهة الافتراضية):** بمجرد فتح التطبيق، يظهر لك نموذج إضافة عملية بيع جديدة لإتمام الطلبات بأقل عدد من النقرات.
2. **سلة مشتريات ذكية مجمعة:** تم دمج المنتجات من المأكولات والمشروبات، شحن رصيد الاتصالات، وألعاب البلايستيشن في مكان واحد.
3. **أزرار التحكم بالأسعار بدرجات 1000 ليرة (Price Steppers):** لتجنب تعب الكتابة على الجوال، يمكنك زيادة أو نقصان الأسعار بنقرات مريحة بمعدل 1000 ليرة سورية، مع إتاحة التعديل اليدوي الكامل.
4. **توزيع الدفع المرن والديون:** يمكنك تحديد المبلغ المقبوض كاش والمبلغ المتبقي كدين. وفي حال وجود دين، يظهر لك حقل اختيار العميل مع إمكانية إضافة عميل جديد فوراً عبر نافذة منبثقة دون الخروج من الشاشة.
5. **الاتصال بالاستضافة الحية (Live Hosting API):** التطبيق مجهز تماماً للاتصال بالـ API المرفوع على استضافتك الحية مع حفظ رمز التوثيق والجلسة تلقائياً في الهاتف.

## هيكل المجلد والملفات المصدرية

تم تنظيم التطبيق باتباع أفضل معايير هندسة الأكواد في Flutter:
- `lib/main.dart` - نقطة الانطلاق وإعدادات المظهر الفاخر ودعم اللغة العربية (RTL).
- `lib/models/` - نماذج البيانات لضمان دقة المعاملات وتوافقها مع الـ API:
  - `cart_item.dart` - نموذج عناصر السلة الموحد (مأكولات، رصيد، لعب بلايستيشن).
  - `product.dart` - نموذج منتجات المخزن.
  - `customer.dart` - نموذج عملاء الديون.
  - `telecom_company.dart` - نموذج شركات الاتصال.
- `lib/services/api_service.dart` - العميل البرمجي للاتصال بالـ API وسحب البيانات وإرسال الفواتير.
- `lib/widgets/` - المكونات البصرية التفاعلية:
  - `glassmorphic_card.dart` - حاوية زجاجية فخمة.
  - `stepper_widget.dart` - أداة التحكم بالأسعار بدرجات الـ 1000 ليرة سورية.
- `lib/screens/` - واجهات التطبيق الرئيسية:
  - `sales_screen.dart` - الشاشة الافتراضية لتسجيل فواتير البيع الموحدة.
  - `settings_screen.dart` - شاشة إعدادات الاتصال بالاستضافة وتسجيل الدخول لحساب الإدارة.

## متطلبات التشغيل والبدء السريع

لتشغيل هذا التطبيق على هاتفك المحمول أو محاكي الأندرويد/الآيفون، اتبع الخطوات التالية:

### 1. تثبيت بيئة عمل Flutter
تأكد من تثبيت بيئة عمل Flutter على جهاز الكمبيوتر الخاص بك. إذا لم تكن مثبتة، يمكنك تحميلها من [الموقع الرسمي لـ Flutter](https://flutter.dev).

### 2. تحميل المكتبات والاعتمادات
افتح نافذة الأوامر (Terminal) داخل مجلد `gamecash-mobile` ونفّذ الأمر التالي لتحميل الاعتمادات المعرفة في `pubspec.yaml`:
```bash
flutter pub get
```

### 3. ربط التطبيق مع استضافة الويب (API)
1. قم برفع مجلد الباك إيند `gamecash-backend` بالكامل على استضافة الويب الخاصة بك.
2. شغّل التطبيق على هاتفك الذكي أو المحاكي باستخدام الأمر:
   ```bash
   flutter run
   ```
3. عند فتح التطبيق لأول مرة، ستظهر لك رسالة تفيد بعدم وجود اتصال بالخادم.
4. انقر فوق زر **"التوجه للإعدادات"** (أو اضغط على أيقونة الإعدادات الزرقاء في الزاوية العلوية).
5. أدخل رابط الاستضافة الكامل المؤدي للباك إيند (مثال: `https://your-domain.com/gamecash-backend`).
6. أدخل اسم المستخدم (`admin`) وكلمة المرور الخاصة بالإدارة.
7. اضغط على **"حفظ وتسجيل دخول"**.
8. مبروك! سيقوم التطبيق بحفظ رمز الولوج (`X-Auth-Token`) بشكل آمن ودائم وسيقوم بسحب قائمة المأكولات والمشروبات الحية، شركات الاتصال المتاحة، وقائمة العملاء، وتفعيل شاشة البيع فوراً!
